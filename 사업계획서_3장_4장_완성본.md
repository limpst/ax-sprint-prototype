## 나. 상용화 대상 제품·서비스 원리 및 우수성

### 1) 핵심 기술 원리

#### ① Y-MaskNet — 비전 AI 손상 탐지 엔진

Y-MaskNet은 에이톰엔지니어링이 독자 개발한 **공공주택 특화 Instance Segmentation 모델**입니다. 일반 Object Detection(Bounding Box)이 "손상이 있는 위치"만 알려주는 데 반해, Instance Segmentation은 **손상 영역의 픽셀 단위 윤곽선(마스크)을 정확히 분리**합니다. 이를 통해 균열 폭(mm), 손상 면적(cm²), 길이(cm) 등 정량적 수치를 직접 산출합니다.

```
드론 / 입주민 앱 이미지 수집
        │
        ▼
[전처리: 해상도 정규화·조명 보정·가명처리(얼굴·번호판 자동 마스킹)]
        │
        ▼
[Y-MaskNet 추론 엔진]
 · Backbone: 공공주택 손상 특화 커스텀 아키텍처 (Mask R-CNN 계열)
 · 탐지 클래스:
   - 수직균열(Vertical Crack)  · 폭 0.2mm 이상 → 즉각 조치
   - 수평균열(Horizontal Crack) · 폭 0.2mm 이상 → 즉각 조치
   - 누수흔적(Water Leakage)   · 습윤면적 200cm² 이상 → 보수
   - 드라이비트 화재취약       · KFS 기준 자동 판별 → 경보
   - 박리·박락(Spalling)       · 폭 0.5mm 이상 → 보수
   - 표면손상(Surface Damage)  · 경과 관찰
        │
        ▼
[Antigravity 오탐 보정 엔진]
 · Y-MaskNet 출력 위에 적층
 · 유사 패턴(그림자·페인트·얼룩) 오탐 필터링
 · False Positive 구조적 제거 → 오탐 0건 목표
        │
        ▼
[정량 산출: 균열폭(mm), 면적(cm²), 위험도 등급(고/중/저)]
        │
        ▼
[XAI 보고서 자동 생성 → FMS/세움터 자동 전송]
```

**핵심 성능 수치:**

| 지표 | 현재 TRL 8 달성값 | 과제 목표(TRL 9) | 측정 기관 |
|------|----------------|----------------|---------|
| Average F1-score | **0.97** | **0.99 이상** | 한국공학대학교 (공식 검증) |
| 최소 탐지 균열폭 | **0.2mm** | **0.2mm 유지** | 실증 환경 기준 |
| 오탐(False Positive) | 5% 미만 | **0건 목표** | 방재시험연구원 검증 |
| 누락(False Negative) | 3% 미만 | **0건 목표** | 방재시험연구원 검증 |

#### ② LIO-SLAM / LiDAR SLAM — 3D Digital Twin 생성 엔진

LIO-SLAM(LiDAR-Inertial Odometry SLAM)은 **GPS 신호가 차단된 도심 협곡·지하 환경**에서도 Velodyne Puck LiDAR 센서와 IMU(관성측정장치)를 6방향 융합하여 건물 전체 3D 점군 맵을 실시간으로 생성합니다.

- **활용:** 건물 전주기 3D 메시 맵핑 → 균열 위치의 정확한 3D 좌표 기록 → 시계열 비교로 손상 진행 속도 추적
- **차별점:** 단순 2D 도면이 아닌 "살아있는 3D 건물 이력"으로 기존 CAD 도면 기반 점검 대비 정보 완전성 100배 향상
- **지반침하 확장:** IoT 경사계·GPR 데이터와 결합 시, 지반 공동(空洞) 조기 탐지 → 싱크홀 예방(v2.0 로드맵 반영)

---

#### ③ 지능형 RPA (Intelligent RPA) — 행정 자동화 엔진

기존 RPA가 단순 반복 클릭 자동화에 머무른 반면, **에이톰-AX의 지능형 RPA는 AI 분류 결과를 실시간 입력 신호로 활용**하여 업무 흐름을 동적으로 자동 처리합니다.

```
[민원 접수 (입주민 앱 / 전화 / 방문)]
        │
        ▼
[Y-MaskNet AI 자동 분류: 유형·위험도 판별 (자동화율 85%)]
        │
        ▼
[지능형 RPA: 담당자 자동 배정 (위험도·지역·전문성 매칭)]
        │
        ├─ 고위험: 즉시 문자 발송 + 구조팀 자동 출동 예약
        ├─ 중위험: 3일 내 현장 점검 자동 일정 생성
        └─ 저위험: 7일 내 처리 예정 자동 통보
        │
        ▼
[FMS REST API: 처리 결과 자동 기록·보고]
```

**자동화율 현황 및 목표:**

| 업무 유형 | 현재 수동 처리 | 에이톰-AX 목표 자동화율 | 연간 절감 효과 |
|---------|-------------|----------------------|-------------|
| 관리비 고지서 발행 | 100% 수동 | **80% 자동** | 월 504건 자동 처리 |
| 계약 만료 알림 발송 | 100% 수동 | **100% 자동** | 연 40건 자동 발송 |
| 민원 담당자 배정 | 100% 수동 | **75% 자동** | 배정 시간 → 즉시 |
| 점검 일정 생성 | 100% 수동 | **90% 자동** | 리드타임 50% 단축 |
| 예방 정비 우선순위 결정 | 전문가 수동 판단 | **AI 자동 스코어링** | 긴급 보수비용 30~40% 절감 |
| **연간 인건비 절감 추정** | — | — | **약 1억원 (1,350만원 시범 확인)** |

#### ④ AI 사전 예방 정비 엔진 — 손상이 사고가 되기 전에 막는 시스템

에이톰-AX의 핵심 철학은 **"사고가 난 뒤 판정하는 것이 아니라, 사고가 나기 전에 AI가 먼저 감지하고 차단한다"** 는 것입니다. XAI(설명가능 AI) 기반 위험도 스코어링 엔진이 다층 데이터를 실시간 종합하여 **손상 진행 속도를 예측하고, 임박한 위험을 사전 경보**합니다.

**7개 예방 지표 및 위험도 기여도:**

| 예방 지표 | 기여도 범위 | 데이터 소스 | 예방 효과 |
|---------|------------|-----------|---------|
| 건물 경과년수(노후도) | 20~32% | 세움터 건축물 도면 API | 노후 단지 점검 주기 자동 강화 |
| 동일 유형 손상 시계열 추이 | 12~25% | FMS 점검 이력 DB | 손상 가속 구간 조기 포착 |
| IoT 센서 이상 감지 누적 횟수 | 10~20% | 에이톰-AX IoT 모듈 | 진동·균열 복합 위험 선제 탐지 |
| AI 손상 탐지 면적·폭 변화율 | 12~22% | Y-MaskNet 정량 출력 | 균열 성장 속도 예측 |
| 마지막 정기점검 경과일수 | 8~16% | 점검 이력 DB | 점검 공백 위험 자동 감지 |
| 기상·계절 취약 패턴 | 4~10% | 기상청 API | 동절기 동결·해동 손상 예측 |
| 드라이비트 면적·위치 | 3~10% | Y-MaskNet 탐지 결과 | 화재 위험 구간 사전 격리 경보 |

**예방 정비 출력물:**
- 위험도 등급(고·중·저) 자동 분류 → 예방 정비 우선순위 목록 PDF 자동 생성
- 단지별 "30일 내 예상 손상 확대 구간" 히트맵 대시보드 표시

### 2) 성과 지표 달성 근거

#### ① F1-score 0.99 달성 가능성

- **현재 기준점:** F1-score 0.97 (공식 검증 완료, TRL 8)
- **0.97 → 0.99 경로:** 실증 데이터 10만 건 기반 Fine-tuning + Antigravity 오탐 보정 엔진 고도화
- **공학적 근거:** 딥러닝 모델은 학습 데이터 규모 10배 증가 시 F1-score 0.02~0.05 향상이 통상적으로 확인됨. 현재 데이터 대비 10만 건은 충분한 개선 여력

#### ② 리드타임 50% 단축 달성 가능성

- **현재 시뮬레이션 확인:** 에이톰-AX 프로토타입(v1.0.0)에서 점검 업무 시간 48.2% 단축 확인
- **실증 목표:** 드론 자동 스캐닝 + AI 즉시 분석 + FMS 자동 전송의 End-to-End 자동화로 50% 목표 달성

#### ③ 매출 25억원 (Year 2) 달성 가능성

- **LH 시장 규모:** LH 관리 공공임대주택 110만 세대 (2025년 기준)
- **동당 600만원 SaaS:** 1% 점유율 = 11,000동 × 600만원 = **660억원 시장**
- **Year 2 목표 25억원 = 시장의 0.04%** → 현실적 달성 가능
- **핵심 전제:** SH 레퍼런스 + 혁신제품 지정 + NET 인증의 3대 인증이 완료되면 LH 조달 진입 장벽 소멸
